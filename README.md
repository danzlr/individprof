# Задание 1

## Файл

`task1.py`

## Описание

В первом задании реализована система поиска ArUco-маркера и остановки робота, когда маркер оказывается строго по центру изображения.

Программа выполняет следующие действия:

1. Получает изображение с камеры.
2. Распознаёт ArUco-маркеры.
3. Определяет положение маркера на изображении.
4. Вращает робота до тех пор, пока маркер не окажется по центру.
5. После центрирования останавливает движение и выводит ID найденного маркера.

## Структура работы

Программа разделена на несколько потоков:

### 1. Поток отправки данных в serial-порт  
Отвечает за передачу управляющих команд роботу.

### 2. Поток работы с камерой  
Считывает кадры и выполняет распознавание маркеров.

### 3. Основной поток  
Запускает остальные потоки, управляет вращением робота и завершает работу при выполнении условия центрирования.

## Основная логика программы

Детекция маркера:

```python
gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
corners, ids, _ = cv2.aruco.detectMarkers(gray, aruco_dict)
```

Определение центра маркера:

```python
if ids is not None:
    cX = int(corners[0][0][:, 0].mean())
    frame_center = frame.shape[1] // 2
```

Вращение до центрирования:

```python
if cX < frame_center - 20:
    send_command(-0.5, 0.5)
elif cX > frame_center + 20:
    send_command(0.5, -0.5)
else:
    send_command(0, 0)
```

# Задание 2

## Файл

`prov5.py`

## Описание

Во втором задании реализовано движение робота вдоль стены с использованием лидара.

Программа получает данные о расстояниях до объектов вокруг робота и на их основе управляет скоростью колёс.

## Основные компоненты программы

### 1. Работа с лидаром  
- Подключение к устройству  
- Чтение пакетов данных  
- Проверка корректности данных  
- Формирование полного сканирования на 360 градусов  

### 2. Связь с роботом  
Через последовательный порт:
- Получение углов и скоростей колёс  
- Отправка управляющих команд  

Частота обмена — 50 Гц.

### 3. Алгоритм управления (motion_control)

Алгоритм:
- Держится выбранной стены (левой или правой)  
- Корректирует направление движения  
- Поворачивает при препятствии впереди  
- Поддерживает безопасную дистанцию

### 4. Считывание маркеров

В отдельном потоке читаются кадры с камеры
При обнаружении маркера, его айди пишется в файл
